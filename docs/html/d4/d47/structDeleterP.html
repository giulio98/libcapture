template<auto deleter>
struct DeleterP< deleter >

Wrapper template for a deleter function that accept a pointer to the struct to free "auto" will accept any signature for the "deleter" function (e.g. void (*deleter)(Struct *)), creating a different struct for each one of them at compile-time
