◆ flush()

void Pipeline::flush ( )

Flush the processing pipelines. If 'async' was set to true when building the Pipeline, the background threads will be completely stopped before the actual flushing. WARNING: this function must be called BEFORE closing the output file with Muxer::closeFile, otherwise some packets/frames will be left in the processing structures and eventual background workers won't be able to write the packets to he output file, throwing an exception
