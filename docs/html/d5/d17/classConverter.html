Converter::Converter ( const AVCodecContext *  dec_ctx,
const AVCodecContext *  enc_ctx,
AVRational  in_time_base,
int  offset_x = 0,
int  offset_y = 0 
)

Create a new audio/video converter. For video, converting the pixel-format and cropping the frame to the output size. For audio, converting the sample-format, sample-rate and channel layout. WARNING: Even if the time-base of the encoder differs from the decoder's one, the timestamps of the frames won't be converted (an eventual conversion will have to be performed separately)

Parameters
dec_ctxthe decoder context containing the input params (time_base will be ignored)
enc_ctxthe encoder context containing the output params (time_base will be ignored)
in_time_basethe time-base of the frames sent to the converter
offset_x(video-only) the horizontal offset to use when performing the cropping of the frames
offset_y(video-only) the vertical offset to use when performing the cropping of the frames
