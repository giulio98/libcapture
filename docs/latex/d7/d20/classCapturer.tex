Start the video \mbox{[}and audio\mbox{]} recording. When this function returns, the capturer internal state will be fully initialized and the recording will be in progress. The video stream will be encoded as H.\+264, with pixel format YUV420, while the audio stream will be encoded as AAC. WARNING\+: if there is already a recording in progress, calling this function will throw an exception. If the specified output file already exists, this function will simply overwrite it without performing any check
